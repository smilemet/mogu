# MOGU!

```모구!```는 공동구매를 보다 쉽게 진행할 수 있도록 총대와 참여자를 이어주는 ```플랫폼 서비스```입니다.

누구나 손쉽게 참여자를 모을 수 있고, 공동구매를 이끌어줄 총대를 찾을 수도 있습니다.

```모구!```는 아래와 같은 서비스를 제공합니다.

- 공구모집 게시판
- 총대찾기 게시판
- 내 구매현황 관리
- 내 공구모집현황 관리
- 공구 참여자 송장 excel로 일괄 내보내기 외 부가기능들
> **Demo** : [현재 개발중...](https://mogu-gonggu.netlify.app)
